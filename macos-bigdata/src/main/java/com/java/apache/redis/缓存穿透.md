
##### 缓存雪崩, 缓存穿透, 缓存击穿
```markdown
缓存穿透: cache和DB中都不没有, 不停的查导致DB压力大; 布隆过虑器解决
缓存击穿: cache有, DB中有; DB压力大; 设置热点数据不过期, 加互斥锁; 指一同条数据
缓存雪崩: 缓存大批量过期, 查询量大导致db压力大; 不同数据
```
