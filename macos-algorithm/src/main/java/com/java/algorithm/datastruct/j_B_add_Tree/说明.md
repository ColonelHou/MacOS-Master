#### B+树特性
```$xslt
1.有k个子树的中间节点包含有k个元素（B树中是k-1个元素），每个元素不保存数据，只用来索引，所有数据都保存在叶子节点。

```
> 每个父节点的数据都出现在子节点上
> 根节点是整个树中最大的元素

```$xslt
$ 卫星数据: 索引元素所指向的数据记录
$ 树中  只有叶子节点有卫星数据, 中间节点仅仅是索引
$ 比B-树更矮胖, IO次数更少

> B-查询性能不稳定, 最优根节点, 最差叶子节点, 而B+每次查询都稳定的
```
